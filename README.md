# EnrollmentApp (.NET 8) - Profesional

Proyecto ASP.NET Core (net8) con:
- Login como página por defecto.
- Gestión de Estudiantes, Programas, Cursos y Horarios.
- Persistencia en archivos JSON (Data/).
- Autenticación por cookies y vistas Razor.

Cómo ejecutar:
1. dotnet restore
2. dotnet run
3. Abrir https://localhost:5001 o http://localhost:5000
